1 การทดลองจำนวนอย่างน้อย 2 การทดลองโดย ที่แต่ละการทดลองต้องระบุ
 1.1 รูปแบบ Reference string ที่ใช้โดยให้อธิบายการ เลือกใช้ค่า ความยาวข้อชุดข้อมูล ลำดับการปรากฎของค่า
 1.2 จำนวน frames ในการทดลอง ต้องมีไม่น้อยกว่า 3 ชุด เช่น 3 frames, 5 frames, 8 frames ให้อธิบายเหตุผลในการกำหนด frame
       ว่าใช้ค่าเหล่านั้นเพื่อ สมมติฐานในการทดสอบว่าอะไร เช่น กำหนดให้มี 3 frames เพื่อให้เห็น ผลของ page faults จาก
       สัดส่วนที่ต่างกันมากระหว่าง reference string จำนวน 50 ตำแหน่ง เป็นต้น
 1.3 ให้ทำการทดลอง โดยใช้ FIFO, Optimal และ LRU algorithm.
2. สรุปผลการทดลอง พร้อมวิเคราะห์ผลแต่ละการทดลอง ว่าน่าจะได้ผลดังนั้นเพราะเหตุใด
3. Copy code ที่ใช้ในการทำงาน โดยให้ แทรกคำอธิบายแนวคิดให้การเขียนสำหรับแต่ละ algorithm
